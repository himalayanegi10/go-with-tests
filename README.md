## Check Version Used
Go - 
```bash
go version
```

Go Lint -
```bash
golangci-lint --version
```

## Installation
Go - Download and install go from [here](https://go.dev/doc/install)
Godoc - 
```bash
go install golang.org/x/tools/cmd/godoc@latest
```
Launch godoc by -
```bash
godoc -http :8000
```
then visit [localhost:8000/pkg/](http://localhost:8000/pkg/) to access Go docs.
